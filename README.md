# Projeto_3 - William Axel

## Estrutura do Projeto
O Projeto_3 contém os seguintes diretórios:

- `Src/` : Código-fonte em C++.
- `Docs/` : Diagramas UML e documentação do projeto.
- `test/` : Testes automatizados.
- `README.md` : Este arquivo com informações do projeto.

## Descrição
Este projeto implementa as funcionalidades conforme instruções do professor. Inclui:

- Classes e métodos documentados.
- UML representando a estrutura de classes.
- Estrutura de diretórios organizada para submissão.

## Instruções de Execução
1. Compile o código com `g++` ou seu compilador de preferência.
2. Execute o programa principal (`main`) a partir do diretório `Src/`.
3. Execute testes contidos no diretório `test/`.
